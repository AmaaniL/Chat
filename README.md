
# Chat app

This is a Chat app made as an excercise.
## Tech Stack

**Client:** React, Tailwind, Javascript, CSS, HTML

**Server:** Java, WebSockets, STOMP


## Deployment

To deploy this project run

```bash
  cd Chatserver
  mvn spring-boot run
```

```bash
  cd Chatclient
  npm run start
```
